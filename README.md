# Todo List Application with React

This project is a Todo List application developed using React.js. It allows you to manage a list of tasks (todos) with features such as adding, deleting, marking as completed, and searching todos easily.


## Features

Add Todos: Quickly add new tasks to your list.
Delete Todos: Remove tasks from your list.
Complete Todos: Mark tasks as completed.
Search Todos: Easily search through your tasks.

## Technologies Used

React.js: JavaScript library for building user interfaces.
CSS: For styling the application.
JavaScript: Main programming language used.

## Installation

1.Clone the repository:
 git clone https://github.com/juanjua1/juanjua1-Todos-react.git

2. Navigate to the project directory:
 cd juanjua1-Todos-react

3.Install the dependencies:
 npm install 

## Running the Project
 To start the development server, run:
 npm start

 This will start the development server, and you can view the project in your browser at http://localhost:3000.
